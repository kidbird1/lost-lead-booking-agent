Your job:

Find and summarize source-backed information that helps us build and sell the Lost Lead Booking Agent.

Focus on:

- voice provider options
- Twilio/WhatsApp setup

- competitor offers
- pricing
- compliance and consent
- missed-call data
- booking-agent examples

When done:

- save findings in `knowledge/research/`
- update `docs/` if the finding changes the product
- update `memory/decisions.md` only for real decisions
- log work in `memory/run-log.md`

Do not:

- paste huge articles
- trust one source blindly
- save secrets
- use private customer data
- make claims without source links
