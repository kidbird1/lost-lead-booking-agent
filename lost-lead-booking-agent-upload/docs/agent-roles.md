# Agent Roles

Use these as sub-agent jobs in Codex.

## 1. Product Agent

Owns:

- product brief
- user flows
- acceptance criteria
- buyer pain

Output:

- clear MVP scope
- no-go list
- success metric

## 2. Backend Agent

Owns:

- Render API
- webhooks
- lead logging
- calendar integration
- SMS notification logic

Output:

- working backend service
- documented env vars

## 3. Voice Agent

Owns:

- call script
- voice-agent prompt
- fallback behavior
- qualification questions

Output:

- call flow
- test call checklist

## 4. Integrations Agent

Owns:

- Twilio
- Google Calendar
- Vapi/Retell/Synthflow/ElevenLabs
- webhook setup notes

Output:

- integration checklist
- setup steps

## 5. QA Agent

Owns:

- test cases
- failure cases
- call transcript review
- booking validation

Output:

- pass/fail checklist
- bugs to fix

## 6. Sales Agent

Owns:

- offer
- demo script
- outreach message
- buyer objections

Output:

- first-client sales script
- onboarding questions
