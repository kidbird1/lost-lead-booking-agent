# Outreach List Template

Use this for the first sales list.

## Columns

```text
Business name
Owner name
Website
Phone
City
Niche
Google rating
Review count
Runs ads
After-hours calls likely
Notes
Outreach status
```

## Best First Prospects

Look for:

- roofers or HVAC companies
- owner-operated
- visible phone number
- active Google Business Profile
- signs they run ads
- no obvious 24/7 receptionist
- average job value high enough to pay for one recovered call

## First List Size

Start with 25 businesses in one city.

Do not build a huge list first.

## Outreach Status

```text
Not contacted
Sent message
Replied
Booked demo
Not interested
Follow up later
```
