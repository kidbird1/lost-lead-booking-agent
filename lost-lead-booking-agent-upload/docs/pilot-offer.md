# Pilot Offer

## Simple Promise

```text
You miss calls. We answer them. You get more booked jobs.
```

## Pilot

```text
30-day missed-call recovery pilot.
```

Includes:

- AI call answering agent
- Lead capture
- Owner WhatsApp/SMS alerts
- Private lead viewer
- Basic issue monitoring
- Manual onboarding

## Suggested Pricing

For first pilots:

```text
Setup: $0-$500
Monthly: $300-$800
```

Use lower pricing for early proof and testimonials.

## Success Metric

```text
Booked or saved leads from calls that would have been missed.
```

## Not Included In Base Pilot

- CRM sync
- Live calendar booking
- Google Maps routing
- Customer WhatsApp agent
- Billing portal

These can become paid add-ons after the core call flow is stable.
