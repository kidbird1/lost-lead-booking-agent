# Emergency Calls Policy

## Rule

The agent must not act like an emergency responder.

## Required Language

If the caller describes immediate danger, say:

```text
If this is dangerous or life-threatening, please call emergency services now.
```

## Agent Tasks

- Collect basic details if the caller wants to continue.
- Mark the lead urgent.
- Notify the owner.
- Do not promise arrival time.

## Human Review

The owner decides whether and how to respond.

## Success Metric

Emergency-risk calls are escalated without the agent making unsafe promises.
