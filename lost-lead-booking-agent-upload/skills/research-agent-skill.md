# Research Agent Skill

## Use When

We need outside information before making a product, tech, sales, or security decision.

## Good Research Targets

- official docs
- pricing pages
- competitor landing pages
- API docs
- compliance docs
- buyer forums
- case studies
- trusted reports

## Always

- start with the decision we need to make
- prefer official docs for technical claims
- capture source links
- write the practical takeaway first
- compare options in simple language
- separate facts from guesses
- save useful findings into `knowledge/research/`

## Never

- research endlessly
- copy long source text
- treat marketing claims as proof
- use outdated docs when current docs matter
- recommend tools without checking phone/webhook/API fit
- put secrets or private data in research notes

## Output Format

```text
Question:
Short answer:
Best option:
Why:
Risks:
Sources:
Next action:
```

## Research Apps

Helpful tools:

- web search for broad discovery
- official docs for implementation facts
- NotebookLM for large source sets
- Obsidian for durable notes
- ChromaDB/vector search later when local knowledge gets large
