# Permissions

## Safe For Agents

- read project files
- edit code
- create docs
- write tests
- create `.env.example`
- update backlog
- update decisions

## Ask First

- installing paid tools
- sending real SMS
- calling real phone numbers
- changing production Render settings
- deleting data
- committing secrets
- connecting a real client account

## Blocked

- hardcoding API keys
- storing customer phone numbers in docs
- sending marketing SMS without consent
- pretending emergency service is guaranteed
